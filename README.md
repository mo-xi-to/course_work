# Анатомический ассистент на базе онтологии FMA

## Основные возможности
- Гибридный поиск: идентификация сущностей по названию и FMAID.
- Многошаговый агент: автоматическое построение цепочек запросов.
- Бенчмаркинг: встроенный инструмент для автоматического сравнения различных моделей.

## Структура репозитория
- `fma_db.py` - модуль загрузки базы FMA и фильтрации пустых атрибутов.
- `tools.py` - описание 4-х инструментов поиска.
- `llm_agent.py` - ядро агента с рекурсивной логикой рассуждения.
- `benchmark.py` - скрипт автоматизированного тестирования.
- `config.py` - настройки моделей и путей.
- `FMA.csv` - файл базы данных.
