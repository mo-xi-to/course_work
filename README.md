# Анатомический ассистент FMA

## Основные возможности
- Идентификация сущностей по названию и FMAID.
- Автоматическое построение логических цепочек (до 15 шагов).
- Алгоритм восполнения пробелов в данных (навигация «от частей к целому»).
- Встроенный бенчмарк для проверки точности.

## Структура проекта
- `llm_agent.py` - ядро системы, логика работы агента.
- `fma_db.py` - поиск в базе данных FMA и алгоритмы навигации.
- `tools.py` - набор из 5 специализированных инструментов для ИИ.
- `config.py` - конфигурация путей и настроек моделей.
- `app.py` - запуск веб-интерфейса (Gradio).
- `benchmark.py` - скрипт автоматизированного тестирования.

## Установка и запуск

### 1. Подготовка окружения
```bash
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### 2. Настройка ключей
Создайте файл `.env` в корне проекта и добавьте

```
YANDEX_KEY=ваш_ключ
YANDEX_FOLDER_ID=ваш_id_каталога
```

### 3. Запуск 
- Веб-интерфейс: `python app.py`
- Консольный режим: `python llm_agent.py`
- Тестирование: `python benchmark.py`
